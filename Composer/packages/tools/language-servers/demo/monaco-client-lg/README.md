# lsp-lp-example

## go to Composer directory
```
yarn install
```

```
yarn build
```


## In Demo/monaco-client-lg directory, Install
```
yarn install
```

## How to start

#### 1. Start language client
```
yarn run start:client
```

#### 2. Start language server
```
yarn run start:server
```

go to [http://localhost:8080/]()
